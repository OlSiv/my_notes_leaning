создать непревилегированную запись в linux и всю работу производить под ней -
Создание непривилегированного пользователя в Linux выполняется через терминал командами -
--
sudo adduser имя_пользователя (рекомендуется)
---- 
или 
--
sudo useradd -m имя_пользователя
----
Новая учетная запись по умолчанию не имеет прав root и доступа к sudo, что обеспечивает безопасность системы. После создания обязательно задайте пароль командой -
--
sudo passwd имя_пользователя
----

основной аккаунт (ноутбук Lenovo) - 
- w
- 7......

созданный аккаунт без привелегий (ноутбук Lenovo) - 
- wo
- 8.......

убедиться, что пользователь был создан, вы можете использовать следующую команду:
--
cat /etc/passwd | grep username
----

выйти из текущей сессии и войти в систему как новый пользователь (переключаться между пользователями linux) -
--
su - username
----

После входа в систему как новый пользователь вы можете проверить, что у вас нет прав суперпользователя, выполнив команду:
--
sudo -l
----

для переключения на текстовую консоль - CTRL + ALT + F1

для переключения обратно на оболочку - ALT + F7, ALT + F8, ALT + F9
