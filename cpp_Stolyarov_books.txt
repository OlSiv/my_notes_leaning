создать непревилегированную запись в linux и всю работу производить под ней -
Создание непривилегированного пользователя в Linux выполняется через терминал командами -
--
sudo adduser имя_пользователя (рекомендуется)
---- 
или 
--
sudo useradd -m имя_пользователя
----
Новая учетная запись по умолчанию не имеет прав root и доступа к sudo, что обеспечивает безопасность системы. После создания обязательно задайте пароль командой -
--
sudo passwd имя_пользователя
----

основной аккаунт (ноутбук Lenovo) - 
- w
- 7......

созданный аккаунт без привелегий (ноутбук Lenovo) - 
- wo
- 8.......

убедиться, что пользователь был создан, вы можете использовать следующую команду:
--
cat /etc/passwd | grep username
----

экранировать символ в командной строке можно с помощью обратного слеша перед символом [\]

если в именах файла есть пробелы, то нужно либо переименовать их чтобы не было пробелов, либо заключить в одинарные ['...'] или двойные ["..."] кавычки 

но внутри двойных кавычек сохраняются 3 символа - 
!
$
` (обратный апостроф)

чтобы провести одну операцию над несколькими файлами, интерпретатор поддерживает подстановку имен файлов по заданному шаблону. В качестве шаблона интерпретатор рассматривает любое слово, не заключенное в кавычки или апострофы, содержащие хотя бы один символ "*" или "?", квадратные или фигурные скобки 

в шаблоне - 
- ? - считается соответствующим одному произвольному символу 
- * - считается соответствующим произвольной цепочке символов (в том числе и пустой)

остальные символы в шаблоне означают сами себя 

квадратные скобки в шаблоне позволяют обозначить любой символ из заданного множества 

символ "!" позволяет обозначить любой символ, кроме указанного - 
--
[!_]*.txt 
----
- любые имена файлов, имеющих расширение "txt", за исключением начинающихся с "_"

фигурные скобки в шаблонах означают любую ццепочку символов из явно перечисленных -
--
*.{jpg,png,gif}
----

процесс в Linux - это программа, запущенная в настоящий момент, при этом запущенная программа может запустить ещё несколько процессов 

посмотреть список процессов -
--
ps 
----

список всех существующих процессов - 
--
ps ax 
----

список всех существующих процессов с информацией о владельцах - 
--
ps axu 
----

список наиболее активных процессов, обновляется раз в секунду -
--
top
----
- выйти из top - q

если процесс не прерывается, то можно убить его так - 
--
kill -9 [номер процесса, посмотреть командой - ps ax]
----
или так -
--
kill -KILL [номер процесса, посмотреть командой - ps ax]
----

чтобы запустить задачу в фоновом режиме, нужно после команды добавить & - например так - 
--
updatedb &
----

чтобы посмотреть задачи в фоне - 
--
jobs 
----

одна программа может одновременно работать с несколькими потоками ввода-вывода, различая их по номерам, которые называют "дескрипторами"
