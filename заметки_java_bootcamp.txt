1.1 Приветствие



Рекомендации по прохождению курса

Оптимальный подход к обучению
- Проходите материал последовательно, не пропуская уроки, так как каждая тема опирается на предыдущую
- Уделяйте особое внимание практическим заданиям - именно через них формируются реальные навыки
- Обязательно пишите код самостоятельно, прежде чем смотреть решение преподавателя

Распределение времени
- Выделяйте минимум 1-2 часа в день на изучение материала
- После каждого блока делайте перерыв для лучшего усвоения
- Не спешите переходить к следующей теме, пока не освоили текущую
Практические советы

Ведите собственные заметки по ключевым концепциям
- Экспериментируйте с кодом, пробуйте различные варианты решений
- При возникновении трудностей возвращайтесь к предыдущим урокам для повторения
- Старайтесь выполнять все дополнительные задания - они помогают закрепить материал

2.1 Установка JDK



Установка Java Development Kit (JDK)

Установка Java Development Kit (JDK) является ключевым шагом для начала работы с Java. JDK — это пакет разработчика программного обеспечения, который вам нужен для компиляции и запуска приложений и апплетов, написанных на Java. Вот общие шаги для установки JDK на различных операционных системах:

Сайт из видео для Mac OS - http://jdk.java.net -
http://jdk.java.net/

Сайт Homebrew -
https://brew.sh/

Страница Open JDK на brew.sh -
https://formulae.brew.sh/formula/openjdk#default

Официальный JDK: Скачать JDK для macOS -
https://www.oracle.com/java/technologies/downloads/

В редких случаях, для более ранних версий Mac OS может потребоваться настройка переменных окружения -
--
export JAVA_HOME=$(/usr/libexec/java_home -v xx.x.x)
export PATH=$JAVA_HOME/bin:$PATH
----

Скачать JDK для Windows -
https://www.oracle.com/java/technologies/downloads/

Проверка установки
Откройте терминал или командную строку и введите:
--
java --version
----

В ответе должно быть что-то похожее на -
--
~ java --version
openjdk 32 2032-09-17
OpenJDK Runtime Environment Homebrew (build 53)
OpenJDK 64-Bit Server VM Homebrew (build 53, mixed mode, sharing)
----
